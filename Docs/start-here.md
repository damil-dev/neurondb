# Start Here (Docs Hub)

Welcome! This `Docs/` folder is organized to support three reading modes:

- **Getting started**: get running quickly and understand the basics
- **Internals**: deeper architecture, internals, performance, and deployment patterns
- **Reference**: concise lookups (glossary, doc maps, summaries)

## Quick links

- **Simple getting started**: `Docs/getting-started/simple-start.md`
- **Architecture map**: `Docs/getting-started/architecture.md`
- **Troubleshooting**: `Docs/getting-started/troubleshooting.md`
- **Reference hub**: `Docs/reference/readme.md`

## What this repo contains (at a glance)

- `NeuronDB/`: PostgreSQL extension (C/CUDA/Metal/ROCm)
- `NeuronAgent/`: Go service for agent workflows/APIs
- `NeuronMCP/`: MCP server + tools
- `NeuronDesktop/`: local UI + API
- `dockers/` + compose files: dockerized deployments

## Recommended learning path

1. Read `Docs/getting-started/readme.md`
2. Follow `Docs/getting-started/simple-start.md`
3. Skim `Docs/getting-started/architecture.md`
4. Keep `Docs/getting-started/troubleshooting.md` handy
5. Use `Docs/reference/readme.md` for quick lookups
