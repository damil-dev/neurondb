# NeuronDesktop

NeuronDesktop is a local management UI and API.

## Where to look in the code

- API service: `NeuronDesktop/api/`
- Frontend: `NeuronDesktop/frontend/`

## Docker

- Compose services: `neurondesk-api`, `neurondesk-frontend`
- From repo root: `docker compose up -d neurondesk-api neurondesk-frontend`

## Ports

- API: `http://localhost:8081/`
- UI: `http://localhost:3000/`
