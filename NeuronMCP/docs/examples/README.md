# NeuronMCP Examples

This directory contains example client implementations and interaction transcripts for NeuronMCP.

## Contents

- [Example Transcript](example-transcript.md) - Complete MCP protocol interaction transcripts demonstrating various workflows

## Example Transcript

The [example-transcript.md](example-transcript.md) file contains detailed examples of:

1. **Vector Search Workflow** - Complete workflow from initialization to vector similarity search
2. **ML Model Training** - Training a machine learning model and checking status
3. **Hybrid Search with Reranking** - Performing hybrid search and reranking results
4. **Error Handling** - Examples of error responses and handling

Each transcript includes:
- JSON-RPC 2.0 protocol messages
- Request/response examples
- Parameter details
- Error handling scenarios
